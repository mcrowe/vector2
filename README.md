# Vector2

2d vector library for Javascript/Typescript

## Usage

> npm install @mcrowe/vector2 --save

## Development

Install npm modules:

> npm install

Run tests:

> npm test

Build:

> npm run build

Publish to npm:

1. Update the version in `package.json`

2. Build using `npm run build`

3. Publish using `npm publish --access=public`
